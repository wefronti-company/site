# Projetos Portfólio

Projetos de portfólio da Wefronti, publicados em subdomínios separados do site principal.

| Projeto   | Subdomínio              | Pasta    |
|----------|-------------------------|----------|
| Fibracom | fibracom.wefronti.com   | fibracom |
| Clarus   | clarus.wefronti.com     | clarus   |

Cada pasta é uma aplicação Next.js independente.

**Deploy no subdomínio (ex.: Vercel)**  
1. Novo projeto → repositório deste repo.  
2. **Root Directory:** `projects-portfolio/fibracom` (Fibracom) ou `projects-portfolio/clarus` (Clarus).  
3. **Domains:** adicione o subdomínio (ex.: `fibracom.wefronti.com`, `clarus.wefronti.com`).  
4. No DNS, crie um CNAME apontando para o valor indicado pelo Vercel.
