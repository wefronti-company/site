# Projetos Portfólio

Projetos de portfólio da Wefronti, publicados em subdomínios separados do site principal.

| Projeto | Subdomínio          | Pasta  | Stack         |
|---------|---------------------|--------|---------------|
| Finora  | finora.wefronti.com | finora | HTML, CSS, JS |

Site estático em HTML, CSS e JavaScript (soluções financeiras).

**Deploy no subdomínio (ex.: Vercel)**  
1. Novo projeto → repositório deste repo.  
2. **Root Directory:** `projects-portfolio/finora`.  
3. **Domains:** adicione o subdomínio (ex.: `finora.wefronti.com`).  
4. No DNS, crie um CNAME apontando para o valor indicado pelo Vercel.
