# Fibracom — Portfólio Wefronti

Case de provedor de internet fibra. Publicado em **fibracom.wefronti.com**.

## Desenvolvimento

```bash
npm install
npm run dev
```

Abre em [http://localhost:3002](http://localhost:3002).

## Deploy (Vercel)

1. Importe o repositório e defina **Root Directory** como `projects-portfolio/fibracom`.
2. Em **Settings → Domains**, adicione `fibracom.wefronti.com`.
3. Deploy.

## Estrutura

- `pages/index.tsx` — landing do case Fibracom.
- Edite conteúdo e estilos conforme o design do case.
